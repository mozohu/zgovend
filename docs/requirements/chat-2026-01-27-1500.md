# 需求討論紀錄（2026-01-27 15:00）

## 客服回覆範圍
- 消費者可透過 LIFF 發起客訴問題、查詢本人問題單與處理過程。
- 客服回覆由「客服系統」（電腦網頁管理系統）處理與回覆，不屬於 LIFF 專案範圍。
- LIFF 需提供問題單詳情與客服回覆內容的查詢。

## 庫存預約設定（Pre-Replenish）
- 定義：由管理員在巡補前建立的「庫存預約設定」，供巡補員現場一鍵套用以加速作業。
- 設定內容包含：貨道商品、滿倉量、庫存量。
- 不包含：清空、補滿等巡補人員專用功能。
- 設定檔特性：可命名、建立複本、刪除、更名。
- 建立方式：可從空白逐一設定，或從指定機台的現有庫存建立。
- 巡補員流程：於巡補中選擇「庫存預約設定清單」→ 套用 → 結束巡補後才寫入本機庫存。

## 撿貨（Picklist）彙總規則
- 機台清單可多選，作為庫存總量的篩選條件。
- 將多台機台的庫存總量依商品分組彙總，計算：
  - 總庫存量
  - 總滿倉量
  - 總缺補量
