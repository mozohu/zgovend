# GUI 文件索引

本資料夾整理 GUI 相關的流程與線框，請依角色與功能查找。

## Sitemap
- `docs/gui/gui-sitemap.mmd`：GUI sitemap（含角色入口、功能關聯與操作深度）
  - 註冊/登入為起始化流程，不計入層數限制。
  - 深度限制從「角色入口/功能列」後開始計算。
  - 為避免線條交叉，重複子頁在各功能下獨立繪製，頁面深度不超過 3 層。
